Tests the 1-sum of char matrices for total unimodularity.

Returns true if and only if the 1-sum of the given components is TU.

If pdecomposition is not NULL and the algorithm has to test regularity of the support matrix, then *pdecomposition will point to a decomposition tree for which the caller must use TUdecFree to free memory. It is set to NULL in case regularity of the support matrix does not need to be determined.

If psubmatrix is not NULL and the matrix is not TU, then a submatrix with an absolute determinant larger than 1 will be searched, which may cause extra computational effort. In this case, *psubmatrix will point to this submatrix for which the caller must use TUsubmatFree to free memory. It is set to NULL otherwise.

Tests a double matrix for total unimodularity.

Returns true if and only if matrix is TU, where we consider entries to be equal to -1, 0 or +1 if their distance is at most epsilon.

If pdecomposition is not NULL and the algorithm has to test regularity of the support matrix, then *pdecomposition will point to a decomposition tree for which the caller must use TUdecFree to free memory. It is set to NULL in case regularity of the support matrix does not need to be determined.

If psubmatrix is not NULL and the matrix is not TU, then a submatrix with an absolute determinant larger than 1 will be searched, which may cause extra computational effort. In this case, *psubmatrix will point to this submatrix for which the caller must use TUsubmatFree to free memory. It is set to NULL otherwise.
