Ensures that signs of sequentially connected char matrix nonzeros qualify for being TU.

The matrix is assumed to be ternary. If sign changes are necessary, only matrix is modified. In particular, transpose is remain unchanged. Expects matrix to have the additional rowStart entry (

See also
TODO).

The function returns 0 if the signs were already correct. Otherwise, it returns 'm' or 't' depending on whether matrix or transpose was modified (or would have been modified).

If submatrix is not NULL and sign changes are necessary, then a submatrix with determinant -2 or +2 is stored in *psubmatrix and the caller must use TUsubmatFree free its memory. It is set to NULL if no sign changes are needed.
