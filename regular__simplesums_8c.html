Checks for a simple 2-sum or 3-sum decomposition of a given ternary or binary linear matroid.

Assumes that decomposition contains valid matrix and transpose entries. A simple 2-sum is one where one component consists of only one row and one column. It is found by splitting off unit vectors.

Returns
Remaining submatrix.
