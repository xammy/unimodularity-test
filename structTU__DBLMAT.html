Detailed Description

Row-wise representation of sparse double matrix.

The columns and values of all nonzeros are stored in entryColumns and entryValues, respectively. Those of row r are stored from rowStarts[r] until (but not including) rowStarts[r+1]. The last row is an exception, since rowStarts[numRows] need not be defined. For convenience, one may store this additional entry. In particular TUdblmatCreate allocates sufficient space for it. However, all public methods use numRows to determine the last row's number of nonzeros via numNonzeros.
